Selectors Elementals

Selectors Universals

És un asterisc, s'ha de utilitzar en certs casos molt concrets, i anar molt en compte

            * {
                background-color: blue;
            }

S'ha de evitar:

            * {
                margin: 0;
                background-color: lightcoral;
            }

Selector de tipus o etqiueta

És simplement posar el nom de l'etiqueta que vlem donar estil

            h1 {
                background-color: lightcyan;
            }

Posar el nom de l'etqiueta que volemn donar estil

            p {
                background-color: lightgreen;
            }

Si volem canviar TOTES les etiquetes <p>

Aquests selectors s'apliquen en la resta d'estils que es faservr per començar una pàgina. Algunes etiquetes com h1 o p per exemple potren uns estils associats per defecte. Aqui els estils es poden restejar.

Per exemple l'etqiueta h1 porta aquests estils

            h1 {
                display: block;
                font-size: 2em;
                margin-top: 0.67;
                margin-bottom: 0.67em;
                margin-left: 0;
                margin-right: 0;
                font-wight: bold;
            }

La majoria d'aquest estils no els coneixem encara, però restrejar o normalitzar els estils h1 voldria dir treure aquests propietats associades a <h1>. El reset o la normalització d'estils es fa a totes les etiquetes. ho veurem més endevant. Podem donar estils a moltes altres etiquetes, com per ecemple <ul> o a <li> i tants d'altres.
